# 🤖 TinyLlama Agents with Ollama

A collection of beginner-friendly AI agents built using **Ollama** and **TinyLlama**.

## 📂 Files

| File                       | Description                              |
| -------------------------- | ---------------------------------------- |
| `basic_chatbot.py`         | Simple chatbot using TinyLlama           |
| `chatbot_with_memory.py`   | Chatbot that remembers previous messages |
| `react_agent.py`           | ReAct-style reasoning agent              |
| `react_simulation_tool.py` | ReAct agent with a calculator tool       |
| `system_prompt.py`         | TechBot using a custom system prompt     |
| `tool_using_Agent.py`      | Agent capable of using multiple tools    |
| `Test.py`                  | Simple test script                       |

## 🚀 Requirements

* Python 3.10+
* Ollama

Install Ollama and pull TinyLlama:

```bash
ollama pull tinyllama
```

Install the Python package:

```bash
pip install ollama
```

## ▶️ Running Examples

Run the basic chatbot:

```bash
python basic_chatbot.py
```

Run the chatbot with memory:

```bash
python chatbot_with_memory.py
```

Run the ReAct agent:

```bash
python react_agent.py
```
